# flowify

**Declarative JSX components replacing React imperative patterns.**

flowify provides a set of React components that replace common imperative patterns
(early returns, ternary operators, `useRef` + `useEffect` boilerplate) with clean, declarative JSX.

## Installation

```bash
pnpm add flowify
```

## Quick Example

```tsx
import { Guard, Show, Switch, Each } from "flowify";

// Before: imperative null check
{user && <Profile name={user.name} />}

// After: declarative with type narrowing
<Guard when={user} fallback={<Login />}>
  {(u) => <Profile name={u.name} />}
</Guard>
```
