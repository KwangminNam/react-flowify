# Use\<T\>

Unwraps a Promise using React 19's `use()` hook and passes the resolved
value to a render-prop.

<Callout type="warning">
Must be wrapped in `<Suspense>` to show a fallback while the promise is pending.
Requires **React 19+**.
</Callout>

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `promise` | `Promise<T>` | ✅ | The promise to unwrap. |
| `children` | `(data: T) => ReactNode` | ✅ | Render function receiving the resolved value. |

## Usage

<Tabs items={['After (flowify)', 'Before (imperative)']}>
<Tabs.Tab>
```tsx
import { Suspense } from "react";
import { Use } from "flowify";

const userPromise = fetchUser(id);

<Suspense fallback={<Spinner />}>
  <Use promise={userPromise}>
    {(user) => <UserCard name={user.name} />}
  </Use>
</Suspense>
```
</Tabs.Tab>
<Tabs.Tab>
```tsx
const [user, setUser] = useState(null);
const [loading, setLoading] = useState(true);

useEffect(() => {
  fetchUser(id).then((data) => {
    setUser(data);
    setLoading(false);
  });
}, [id]);

if (loading) return <Spinner />;
return <UserCard name={user.name} />;
```
</Tabs.Tab>
</Tabs>

## Notes

- The promise should be created **outside** the component to avoid re-creating it on every render.
- Pair with [`<AsyncBoundary>`](/docs/async-boundary) for combined loading + error handling.
