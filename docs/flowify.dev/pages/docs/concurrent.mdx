# Concurrent

Namespace grouping for React **Concurrent Rendering** related components.

Groups [`Deferred`](/docs/deferred) and [`Transition`](/docs/transition) under
a single namespace to clarify their relationship with React's Concurrent features.

<Callout type="info">
Both components are also available as individual imports. `Concurrent` is
a convenience namespace — not a compound component with shared state.
</Callout>

## Usage

```tsx
import { Concurrent } from "flowify";

// Same as <Deferred>
<Concurrent.Deferred value={query}>
  {(deferredQuery) => <SearchResults query={deferredQuery} />}
</Concurrent.Deferred>

// Same as <Transition>
<Concurrent.Transition fallback={<Spinner />}>
  {(isPending, startTransition) => (
    <button onClick={() => startTransition(() => setTab("posts"))}>
      Show Posts
    </button>
  )}
</Concurrent.Transition>
```

## Import styles

```tsx
// Individual imports
import { Deferred, Transition } from "flowify";

// Namespace import
import { Concurrent } from "flowify";
// Concurrent.Deferred, Concurrent.Transition
```
