# Transition

Provides `useTransition()` state as a **declarative render-prop** component.

Gives children access to `isPending` and `startTransition` without
needing a separate hook call.

<Callout type="info">
This is a React **Concurrent** feature. Also available as `Concurrent.Transition`.
</Callout>

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `children` | `(isPending: boolean, startTransition: TransitionStartFunction) => ReactNode` | ✅ | Render function receiving transition state. |
| `fallback` | `ReactNode` | | Rendered instead of children while a transition is pending. |

## Usage

<Tabs items={['After (flowify)', 'Before (imperative)']}>
<Tabs.Tab>
```tsx
import { Transition } from "flowify";

<Transition fallback={<Spinner />}>
  {(isPending, startTransition) => (
    <button
      disabled={isPending}
      onClick={() => startTransition(() => setTab("posts"))}
    >
      Show Posts
    </button>
  )}
</Transition>
```
</Tabs.Tab>
<Tabs.Tab>
```tsx
const [isPending, startTransition] = useTransition();

if (isPending) return <Spinner />;

return (
  <button
    disabled={isPending}
    onClick={() => startTransition(() => setTab("posts"))}
  >
    Show Posts
  </button>
);
```
</Tabs.Tab>
